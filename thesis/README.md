# Thesis Drafts

This folder contains draft material for the thesis on the autonomous tyre inspection robot project.

## thesis_first_draft.md

First draft of the thesis (March 2025). It covers:

- Context: fourth-year placement with the company (Prague), drive-over vs mobile inspection
- Problem and requirements
- Hardware exploration (Galaxy XE, Waveshare, OAK camera, 2D LiDAR limitations)
- Presentation to company (Sept–Dec), viability, provision of SLAMTEC Aurora
- Model training (YOLO, vehicles/people, best_fallback.pt, wheel and vehicle components)
- Hardware setbacks: Jetson rail power failure, Aurora failure, loss of ~3 weeks of uncommitted work
- Aurora diagnostics and replacement; migration to Raspberry Pi 5 and Coral; return to Jetson
- Current architecture: Aurora semantic, Nav2, segment_3d, inspection_manager, tyre order, map reset
- Current state: what works, what is in progress (perfecting autonomous navigation)

British English throughout. No emojis. Professional thesis style. The document is a first draft; the system is not yet fully production-ready.
